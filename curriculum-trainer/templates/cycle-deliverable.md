# Cycle Deliverable Template

Every completed cycle must contain all twelve sections below. Incomplete output
is the most common failure mode of curriculum content — this template prevents
it. Do not skip sections. Do not add sections.

## Section 5 — The Engineered Failure

**Which weakness this cycle exposes:**
[Randomness / Hallucination / Skipped steps / Format drift / Unsafe data handling]

**The specific mechanism:**
[What exact part of the task produces the failure, and in what form. Be precise

**Expected output the learner sees:**
[At least one realistic example of what raw AI produces. Concrete, not

**Why this scenario reliably triggers this failure:**
[The design reasoning. The trainer must be able to defend why this task produces
